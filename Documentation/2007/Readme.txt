Title: Platform Engine v2.4
Author: Martin Piecyk
Last Updated: 2007-02-08

Platform Engine Demo.gm6 - Contains a demonstation of the features of 
the Platform Engine. Open and run this first to see the features of the 
platform engine. 

Platform Engine Tutorial.rtf - Contains a tutorial that teaches you how 
to use the Platform Engine. Read this if you would like to learn how to 
use this Platform Engine in your game. 

Platform Engine Tutorial.gm6 - Used in the tutorial. Does not contain 
any rooms and therefore will not presently run. 

Platform Engine Tutorial - Completed.gm6 - If you complete the tutorial, 
you should end up with a file similar to this one. 

Developer Notes - Documentation for developers who want to use the 
platform engine.
